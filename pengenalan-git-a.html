Pengenalan Git

Kenapa pake Git?

Sebuah Dongeng

Di sesi pertama ini kamu akan kami kenalkan dengan git. Buat kamu yang merasa sudah kenal dengan Git, bisa langsung melanjutkan ke sesi 2. Tapi kami punya sebuah dongeng menarik yang dapat digunakan untuk mengundang teman-teman setimmu untuk mulai menggunakan Git. Baik, sebelum memulai perkenalan Git, perkenankan kami menceritakan sebuah dongeng tentang 4 Mahasiswa dari sebuah universitas di Bandung.

Cerita ini hanya fiktif belaka. Jika ada kesamaan nama tokoh, tempat kejadian ataupun cerita, itu adalah kebetulan semata dan tidak ada unsur kesengajaan.

Dahulu kala, 4 mahasiswa jurusan ilmu komputer bekerja sama membuat aplikasi untuk tugas kuliah mereka. Mereka adalah Siswa (Semester 7), Siswi (Semester 7), Siswo (Semester 9), dan Siswu (Semester 11). Siswa dipilih sebagai ketua kelompok. Tidak ada alasan khusus dibalik dipilihnya Siswa sebagai ketua kelompok. Semua beranggapan karena jika diurut secara alfabetik, Siswa berada di urutan pertama.

Mereka ditugaskan membuat aplikasi sistem informasi pendataan keluarga di kelurahan berbasis web, menggunakan framework CodeIgniter 2. Terkejut mereka mendengar judul tugas ini. Tapi semua bernafas lega setelah dosen menjelaskan bahwa mereka tidaklah harus mendata semua manusia di suatu kelurahan. "Untunglah, kalau benar harus mendatangi semua rumah di kelurahanku, kapan aku lulus?" kata Siswu sambil mengusap-usap dadanya.

Deadline ditetapkan 1 bulan lagi. Siswa tahu benar dia harus memobilisasi tim-nya dari mulai detik ini. Selesai jam kuliah, Siswa langsung mengumpulkan tim-nya untuk rapat perdana. "Ok, Sis, kamu.." belum selesai kalimat dari mulut Siswa, "Iya, Wa?" Siswi, Siswo, dan Siswu menanggapi bersamaan. Siswa baru ingat nama mereka berawalan sama. "Maksudku, Wi.. kita nanti mengerjakan bagian backend, database dan controller." Siswi mengangguk. "Kang Wo, dan Kang Wu.. tolong handle bagian tampilan, frontend, termasuk form untuk entry data. Bagaimana, kang?" Siswo langsung mengiyakan, namun Siswu yang masih sibuk dengan smartphone-nya belum menanggapi. "Kang Siswu, gimana? Handle bagian frontend ya?" lalu disambut dengan santai oleh Siswu "Siap..". "Baiklah, Siswi tolong ambil bahan tugasnya di Pak Dosen. Kita butuh dokumen spesifikasi sistem untuk memulai perancangan database. Kita berkumpul lagi di kampus besok lusa ya." Semua mengangguk sambil bersiap pulang ke kosan masing-masing.

Setelah rapat pertama, tim Siswa hanya lengkap berkumpul 1 kali. Sedangkan deadline pengumpulan tugas tinggal 2 hari lagi. Karena sulit mencocokkan waktu, Siswa seringkali hanya mengerjakan tugas dengan Siswi saja. Itupun karena jadwal kuliah mereka sama. Akhirnya Siswa memutuskan agar tim berkumpul di sebuah cafe di dekat kampus untuk menyelesaikan tugas kuliah ini. Pesan pendek dikirimkan ke semua anggota tim.

"Harap berkumpul di Cafe Bahari pukul 13:00 hari ini, deadline tugas kita tinggal 2 hari lagi. Mohon maaf jika tidak hadir, saya akan melaporkan siapa saja yang tidak berkontribusi di kelompok ini. Terima Kasih."
Pukul 13:15 di Cafe Bahari. "Kang Wo, ini backend-nya sudah hampir selesai. Bagaimana dengan bagian frontend-nya? Oh iya, kang Siswu mana ya?" tanya Siswa. "Oh Siswu sebentar lagi mungkin datang. Ini saya sudah kerjakan bagian tampilan form entry data. Saya email saja ke kamu, Wa?" Siswo sudah siap-siap membuka tab compose new email di browsernya. "Jangan, kang. Di-copy saja ke flashdisk Siswi, semua file dikumpulkan di flashdisk Siswi." Siswi lalu mencabut flashdisk dari laptopnya. "Iya, kang, nanti semua hasil pekerjaan saya yang gabung" Siswa mengamini.

Tim Siswa lalu melanjutkan bagian pekerjaannya masing-masing. Tapi tak lama setelah itu, "Kang Wo, file pekerjaan Kang Siswo yang di folder sispenkelkel.. berweb.. kan?" Raut muka Siswa berubah setelah mendengar Siswi menanyakan hal ini ke Siswo. "Iya, di folder sispenkelkelberweb.. kenapa Wi?" Siswo heran, takut, dan mulai merasa bersalah. "Kenapa Wi? Virus, wi?" Siswa langsung tanggap. "Kayaknya sih, ini soalnya file di flashdisk aku semuanya jadi shortcut..". "Astaga.." Mimpi buruk Siswa jadi kenyataan. "Padahal saya udah pakai SKADAV loh, Wi.." hanya Siswo yang tahu Aplikasi Skadav di komputernya terakhir diupdate 6 bulan yang lalu. "Ya sudah, kang, tolong kirim file hasil pekerjaan Kang Siswo ke email saya di siswa96@mgail.com. Siswi, coba cek file hasil pekerjaan kamu.." Siswa tidak menyelesaikan kalimatnya setelah Siswi memperlihatkan layar laptopnya ke Siswa "Wi..semua file kamu jadi shortcut?" Siswa tidak percaya ini semua terjadi begitu cepat! "Wi, maaf, saya coba cari solusinya di google, barangkali file tugas kuliah kita masih bisa diselamatkan.." Siswo mencoba menebus kesalahannya. "Percuma, kang. Sekarang lebih baik Kang Siswo Kirim file hasil pekerjan akang ke email saya. Saya coba 2 hari ini program ulang semua yang belum digabungkan di komputer saya". Hari itu berjalan sangat lambat. Siswu tak kunjung datang. Siswa dan Siswi akhirnya mengerjakan ulang program bagian backend yang selama ini sudah mereka kerjakan.

Siswa dan Siswi akhirnya berhasil mengerjakan ulang kode mereka yang hilang di rusak virus. Memang tak sesempurna yang dulu, sekadar cukup untuk dikumpulkan, dan dengan harapan dapat nilai cukup untuk lulus. Karena Siswu terhitung ikut berkumpul 2 kali, Siswu tetap mendapatkan nilai dari tugas kuliah ini. Dan benar saja nilai yang didapatkan jauh dari sempurna, bahkan mereka terancam harus mengulang kuliah ini tahun depan. Semua kecewa. Seharusnya semua tidak berakhir seperti ini.
