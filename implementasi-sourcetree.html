Implementasi : Sourcetree

Management Git.

Download dan pasang SourceTree kamu.

Sudahkah kamu menggunakan SourceTree hari ini? haha. Jika kamu belum mengenal SourceTree, aplikasi ini adalah Git dan Mercurial Client yang dibagikan secara gratis hanya untuk Windows dan Mac, jika kamu pengguna GNU/Linux jangan bersedih hati, kamu bisa menggunakan GitEye sebagai Git Clientnya. Download SourceTree kamu di https://www.sourcetreeapp.com/ dan install sesuai instruksi dari SourceTree tersebut.

Setting SourceTree

Jika kamu sudah menginstall SourceTree, sebaiknya kamu mengatur SSH Key yang sudah kamu punya pada sebelumnya. Kamu bisa mengatur SSH Client Configuration dengan file id_rsa.pub yang kamu miliki.

Push

Setelah semua pekerjaan kamu dirasa sudah cukup dan orang lain harus melihat hasil pekerjaan kamu, kamu harus melakukan PUSH. Jadi ibarat kamu ngumpulin tugas, kalau udah beres pasti kamu harus mengirimkannya kan? nah fasilitas PUSH ini adalah bagian proses mengumpulkan tugas tersebut. Jadi nantinya pekerjaan kamu akan tersimpan di Git dan teman kamu dapat melihat dan menggabungkannya.
